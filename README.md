# Flux — Network Security Monitoring Dashboard

Flux is an ongoing cybersecurity project focused on monitoring network traffic, detecting anomalies, and flagging suspicious activity in real time.

Designed as a hands-on learning project, it aims to provide a clear and practical way to understand network behavior and basic threat detection.

---

## 🚀 Features

* Real-time network traffic monitoring
* Detection of unusual patterns and anomalies
* Terminal-style alert feed with timestamped events
* Planned visualization of traffic and activity

---

## 🛠 Tech Stack

* Python
* Linux
* Network Security Tools (Scapy, tcpdump)
* CTF-inspired workflows

---

## 📌 Project Status

**Ongoing**
Currently working on:

* Packet capture implementation
* Basic anomaly detection logic
* Initial alert system

---

## 🎯 Goals

* Build a functional network monitoring tool
* Improve understanding of traffic analysis and detection
* Simulate real-world cybersecurity scenarios
